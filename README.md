## Description

This game is designed to raise awareness about autism. In our game, players move through the environment with sensory zones that stimulate the user. When the player enters these sensory zones, their sensory overload increases. If it becomes too high, the player is reset at the starting point.

Players can reduce their sensory load by entering quiet zones, these allow them to recover their sensory load before continuing. The goal of the game is to reach the exit zone while managing sensory levels.

The game increases difficulty at each level, with the current hardest level being the third level. As difficulty increases, objects and sensory zones are changed to make it more difficult for players to manage the environment, encouraging players to plan their movements and manage their sensory load carefully.

## Interaction Instructions

1. Open the project folder
2. Launch index.html using Live Server
3. Use W/A/S/D or Arrow Keys to MOVE
4. SHIFT to enter FOCUS MODE (reveal path to the exit)
5. Collect ALL of the collectible circles for the green exit zone to appear
